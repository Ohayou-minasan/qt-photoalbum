# 电子相册

## 项目简介

本软件是一个基于 Qt 6 开发的电子相册应用程序，支持图片查看、幻灯片轮播预览以及背景音乐播放。界面简洁，操作方便，适合个人照片浏览或作为演示工具。

## 主要功能

### 1. 图片查看

支持打开常见格式的图片文件，并在主界面中央显示。用户可以通过鼠标或按钮进行上一张、下一张切换。如需缩放和平移，也可通过鼠标滚轮或拖拽实现。

### 2. 幻灯片轮播预览

提供自动轮播模式，类似 PPT 播放。启动后图片将自动切换，适用于展示或观赏场景。轮播过程中可随时暂停或停止。

### 3. 背景音乐播放

支持加载本地音频文件（如 MP3），在浏览图片的同时播放背景音乐。用户可控制播放、暂停、调节音量，不影响图片操作。

### 4. 目录管理

支持选择文件夹，自动加载文件夹内的所有图片，并按文件名排序。同时提供手动切换（上一张/下一张）和直接跳转到指定图片的功能。

## 技术栈

- 开发框架：Qt 6（C++）
- 多媒体支持：Qt Multimedia 模块
- 界面构建：Qt Widgets（结合 Qt Designer）
- 构建工具：CMake

## 编译与运行

编译前请确保已安装 Qt 6 开发环境，并包含 Multimedia 组件。
